
# DND 2024 Manager

## Descripción
Proyecto Node.js para gestionar campañas y contenido de Dungeons & Dragons 2024.

## Requisitos
- Node.js 16+
- npm o yarn

## Instalación
```bash
npm install
```

## Uso
```bash
npm start
```

## Estructura del Proyecto
```
dnd2024-manager/
├── src/
├── node_modules/
├── package.json
└── README.md
```

## Scripts Disponibles
- `npm start` - Inicia la aplicación
- `npm test` - Ejecuta pruebas
